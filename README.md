# ASCII Art Generator 

Convert any image into ASCII art with multiple styles:
- Ultra detailed 70-character ASCII
- Shadow-boosted high contrast
- Matrix hacker-style ASCII
- Emoji pixel-art mode
- Full RGB colored ASCII

## Inversion styles
- Clean modular codebase
- CLI interface
- ANSI Truecolor support
- Works on Linux, macOS, Windows
- Save output to `.txt`
